# Admin Dashboard

The admin dashboard is for the website owner and staff only. There are no third-party seller tools.

## Main Navigation

- Dashboard
- Products
- Categories
- Inventory
- Orders
- Returns
- Refunds
- Customers
- Campaigns
- Coupons
- Gift Cards
- Banners
- Reviews and Q&A
- Reports
- SEO
- Settings
- Audit Logs

## Dashboard Overview

Key cards:

- Today sales
- Total orders
- Pending orders
- Low stock products
- Active customers

Charts:

- Sales over time

- Traffic to conversion funnel

## Product Management

Admin can:

- Create, edit, archive, and publish products.
- Upload multiple images.
- Upload product videos.
- Manage variants.
- Add technical specifications.
- Set warranty, return, and shipping information.
- Assign product type: brand new, clearance, overstock, seasonal discount, liquidation, promotional.
- Bulk import products through CSV or XLSX.
- Export products.

Product form sections:

- Basic information
- Category and brand
- Pricing
- Variants
- Inventory
- Images and videos
- Description
- Specifications
- Shipping, warranty, and return details
- SEO

## Inventory Management

Admin can:

- View available, reserved, and sold stock.
- Adjust stock with reason.
- See stock movement history.
- Configure low-stock threshold.
- Receive low-stock alerts.
- Block checkout for out-of-stock variants.

## Order Management

Admin can:

- View all orders.
- Filter by status, date, customer, payment method, and delivery district.
- Update order status.
- Print packing slips.
- Download invoices.
- Add shipment tracking.
- Send customer notifications.

Order statuses:

- Pending
- Confirmed
- Packed
- Shipped
- Delivered
- Cancelled
- Returned

## Refund and Return Management

Admin can:

- Review return requests.
- Approve or reject returns.
- Record received items.
- Trigger refunds.
- Track refund status.
- Restore inventory where appropriate.

## Discount Management

Admin can:

- Create flash deals.
- Create clearance campaigns.
- Create daily deals.
- Schedule seasonal campaigns.
- Create coupons.
- Create gift cards.
- Set product-specific promotional prices.

## Banner and Content Management

Admin can manage:

- Hero slider banners.
- Promotional banners.
- App promotion banners.
- Brand showcase.
- Homepage section ordering.
- Newsletter text.

## Customer Management

Admin can:

- View customer profile.
- View customer orders.
- Block or reactivate customer accounts.
- View reward points.
- View wishlist and reviews for support purposes.

## SEO Management

Admin can:

- Edit meta title and description.
- Edit Open Graph information.
- Regenerate XML sitemap.
- Preview product schema markup.
- Manage canonical URLs.

## Roles and Permissions

Recommended roles:

- Super Admin: Full access.
- Store Manager: Products, inventory, orders, campaigns.
- Support Staff: Customers, orders, returns, refunds.
- Content Manager: Banners, SEO, homepage content.
- Analyst: Reports only.

All admin accounts should require two-factor authentication.
