# SmartKinmelHub

SmartKinmelHub is a single-vendor ecommerce platform for Nepal with a Daraz-like shopping experience and Amazon-inspired trust, speed, and conversion patterns.

Domain: `smartkinmelhub.com`  
Tagline: `Smart Shopping for Every Nepali`

## Recommended Stack

- Frontend: Next.js, React, TypeScript
- Backend: NestJS, Node.js, TypeScript
- Database: PostgreSQL
- Cache: Redis
- Search: Elasticsearch
- Infrastructure: Docker, CI/CD, CDN, automated backups

## Product Scope

SmartKinmelHub supports only owner-managed products:

- Brand new products
- Stock clearance products
- Overstock inventory
- Seasonal discounts
- Liquidation inventory
- Promotional products

It does not support third-party sellers, auctions, bidding, bargaining, used products, or second-hand products.
